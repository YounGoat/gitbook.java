#	Maven Plugins

>	Maven is - at its heart - a plugin execution framework; all work is done by plugins.
>	https://maven.apache.org/plugins/index.html

##	直接执行插件

```bash
mvn <plugin-prefix>:<goal>
mvn <plugin-group-id>:<plugin-artifact-id>[:<plugin-version>]:<goal>
```
