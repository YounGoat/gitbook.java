#	Java 语法

[^1]: “加州大学”并不是一所大学，而是由一个美国加利福尼亚州十所公立大学组成的系统，所谓的“分校”或“校区”才是真正意义上独立的大学。我们常听说的“加州大学伯克利分校”之于加州大学，某种程度上类似英国之于英联邦。戴维斯建校较晚（1959年），至今不能称百年，学术排名却常年稳居世界百强之列，实力不容小觑。
