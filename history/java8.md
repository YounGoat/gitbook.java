#	Java 8

##	新特性

以下重要特性都是自 Java 8 才开始予以支持的：

*	函数式接口与 &lambda; 表达式
*	接口的默认方法与静态方法；
*	集合的流式操作；

此外，Java 8 进一步丰富了 Annotation，

###	函数式接口

有且只有一个抽象方法（SAM, Single Abstract Method）的接口，即称为函数式接口（Functional Interface）。
